このアプリは以下の課題を実装したものである

トレーディングカードゲームで行われた対戦成績を管理するアプリケーションを作成してください。
GUIによる動作を必須とします。

**[必須実装機能]**

・対戦成績登録機能
自身の使用デッキを選択し、対戦相手のデッキ、先攻後攻などの要素とともに勝敗を登録す
る。

・デッキ登録機能
対戦成績登録機能で選択するデッキを登録する機能。

・戦績表示機能
対戦成績登録機能で登録した対戦結果が見れる機能。
勝率を計算して表示する。

・GUIによる動作

**[入力]**

使用デッキ（必須）

勝敗（必須）

先攻後攻など

**[出力]**

戦績（必須）

勝率（必須）
